# API-USERS

## Description

API Users is a small practice project focused on working with external APIs. 
It fetches user data from a public API, displays it in a table, and allows filtering users by name or city.


## Technologies used

CSS, Bootstrap, React.Js. API end Point https://jsonplaceholder.typicode.com/users

```bash

npm install

# Start the development server
npm run dev
